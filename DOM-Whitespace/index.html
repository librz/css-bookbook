Whitespace in DOM Node
Hello World! 
  Hello  World!
  Hello  World!  

Whitespaces from start & end of text inside HTML tags(not including <pre>) will be trimmed.

Whitespaces in between words will be reduced to one.

If you want to preserve whitespace, either use <pre> or use HTML code for space(&nbsp;)

For more info, see: MDN: How whitespace is handled by HTML, CSS, and in the DOM
